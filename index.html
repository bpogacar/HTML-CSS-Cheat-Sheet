Common CSS Properties

Property Description
color Sets the text color.
background-color Sets the background color.
font-family Specifies the font for text.
font-size Sets the size of the font.
font-weight Defines the thickness of the characters.
line-height Controls the height of lines of text.
width Sets the width of an element.
height Sets the height of an element.
margin Sets the margin space outside the border.
padding Sets the padding space inside the border.
display Specifies the display behavior of an element (block, inline, flex, etc.).
position Sets the positioning method (static, relative, absolute, fixed).
top, right, bottom, left Specifies the positioning of an element.
border Sets the border properties.
border-radius Defines the curvature of border corners.
box-sizing Defines how the width and height of an element are calculated.
outline Sets the outline properties.
display: flex Enables the flexbox layout model.
flex-direction Sets the direction of the flex container.
justify-content Aligns items on the main axis.
align-items Aligns items on the cross axis.
display: grid Enables the grid layout model.
grid-template-columns, grid-template-rows Defines the columns and rows in a grid.
grid-gap Sets the gap between columns and rows.
text-align Sets the horizontal alignment of text.
text-decoration Specifies the decoration added to text.
text-transform Controls the capitalization of text.
@media Defines media queries for responsive design.
